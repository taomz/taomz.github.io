机器学习项目开发流程（中文版）

Posted on 2019-04-10 | Edited on 2019-04-17 | Comments: | Views:

本文翻译自原文 Machine Learning Project Checklist，本文主要指导您大部分机器学习项目的开发流程，主要包括8个步骤：

当然，您可以根据具体需求调整此开发流程。

1. 问题构建

2. 获取数据

注意：尽可能自动化获取新数据。

3. 探索式分析数据

注意：尝试从相关领域专家那里获取有关这些步骤的见解。

4. 准备数据

注意：

  • 在数据副本上操作（保持原始数据的完整性）。
  • 为所有数据编写转换函数，原因是：
    • 当下次获取新数据集时，很方便的准备数据
    • 可以在将来的项目中使用这些转换函数
    • 用来清洗和准备测试数据集
    • 在解决方案生效后，可以用来清洗和准备新的数据实例
    • 将您的准备选择作为超参数时更容易

5. 模型选择

注意：

  • 如果数据量很大，可能需要采样出较小的训练集，以便在合理的时间内训练出多个不同的模型（请注意，这不利于复杂模型，如大型神经网络或随机森林）。
  • 再次强调，尽可能使下面的这些步骤自动化。

6. 模型微调

注意：

  • 这步骤中尽可能多的使用数据，尤其是要进行最终调优的时候。
  • 尽可能自动化。

测试泛化误差后不要调整模型：那会使得模型对测试集过拟合。

7. 展示您的解决方案

8. 上线、监控和维护系统
